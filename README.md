# 📊 Customer Churn Analysis

## 📌 Project Overview
This project focuses on analyzing customer churn to understand the key factors that lead to customer attrition. The goal is to derive actionable insights that help businesses improve customer retention and reduce revenue loss.

---

## 🎯 Objectives
- Analyze customer churn patterns  
- Identify high-risk customer segments  
- Understand impact of contract type, tenure, and payment method  
- Provide data-driven recommendations  

---

## 📂 Dataset Features
- Customer ID  
- Gender  
- Age  
- Contract Type  
- Tenure  
- Internet Service  
- Payment Method  
- Monthly Charges  
- Total Charges  
- Churn (Yes/No)  

---

## 🔍 Key Insights
- Highest churn observed in **Month-to-Month contracts**  
- **Debit Card users** show higher churn  
- **Fiber Optic customers** have higher churn  
- Peak churn in **2–4 year tenure segment**  
- Overall churn rate is approximately **29%**

## 🛠️ Tools & Technologies

---

## 💡 Business Recommendations
- Promote **long-term contracts**  
- Improve **fiber service quality**  
- Encourage **low-churn payment methods**  
- Target **mid-tenure customers** with offers
